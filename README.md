# Challenge Login Frontend Wiser

## Como utilizar localmente

- Ao entrar a na pasta do projeto execute `npm install` ou `yarn` para instalar
os pacotes

- Em seguida, execute o seguinte comando:

```bash
npm run dev
# or
yarn dev
```
